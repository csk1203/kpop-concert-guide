# 应援物品打包清单

## 应援棒套装

### 必带
- [ ] 官方应援棒 x1
- [ ] 7号电池（AAA）x6-9组
- [ ] 电池收纳盒（防止短路）
- [ ] 应援棒保护套/收纳袋

### 小贴士
- 电池建议分开放置，不要预装在应援棒内
- 可多带一个应援棒（以防故障或帮朋友带）

## 手幅与标识

### 必带
- [ ] 团体官方手幅 x1-2
- [ ] 成员个人手幅（主推成员）
- [ ] 折叠扇子（夏季必备）

### 可选
- [ ] LED手幅（检查电池）
- [ ] 反光手幅（夜场效果好）
- [ ] 应援毛巾

## 穿戴配饰

### 服装
- [ ] 团体应援色T恤/卫衣
- [ ] 舒适下装（长裤/长裙）
- [ ] 轻薄外套（场馆空调冷）

### 配饰
- [ ] 头箍/发饰（应援色）
- [ ] 应援色丝带/发绳
- [ ] 团体徽章/胸针
- [ ] 团体同款配饰

### 鞋子
- [ ] 舒适运动鞋（长时间站立）
- [ ] 备用袜子

- [ ] 透明收纳袋（安检用）
- [ ] 润喉糖（大量）

- [ ] 护手霜

### 卫生用品
- [ ] 免洗洗手液

- [ ] 卫生巾（备用）

### 应急物品
- [ ] 创可贴

- [ ] 复数小卡（用于交换）

- [ ] 小卡收纳册

### 应援物
- [ ] 自制小卡/贴纸（用于分发）
- [ ] 糖果/零食（分享用）

## 电子设备

### 必带
- [ ] 手机
- [ ] 充电宝（10000-20000mAh）
- [ ] 充电线
- [ ] 耳机（场外等待时用）

### 可选
- [ ] 小型自拍杆（场外使用）
- [ ] 迷你风扇（夏季）
- [ ] 暖宝宝（冬季）

## 包装建议

### 分层打包

**随身小包（带入场馆）**：
- 应援棒
- 手机
- 充电宝
- 润喉糖
- 小钱包（现金+卡）

**双肩包/大包（寄存或放酒店）**：
- 备用衣物
- 大量零食
- 备用电池
- 笔记本电脑等贵重物品

### 安检友好

- 使用透明袋装小物品，方便安检
- 液体单独装，不超过100ml
- 尖锐物品（剪刀等）放托运行李

## 购买建议

### 提前购买
- 应援棒（官方渠道）
- 特定手幅/周边
- 电池（国内买更便宜）

### 韩国当地购买
- T-money卡
- 零食饮料
- 紧急日用品
