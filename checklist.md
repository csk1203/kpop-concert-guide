# 韩国演唱会行前检查清单

使用本清单在出发前逐项确认，确保无遗漏。

## 证件与文件

- [ ] 护照原件（有效期6个月以上）
- [ ] 韩国签证（或确认免签资格）
- [ ] 演唱会门票（电子票保存至手机/实体票）
- [ ] 机票行程单（电子版即可）
- [ ] 酒店预订确认单
- [ ] 身份证（国内段可能需要）
- [ ] 护照复印件（备用）
- [ ] 紧急联系人信息卡片

## 金融与支付

- [ ] 韩元现金（建议换50-100万韩元）
- [ ] Visa/Mastercard信用卡
- [ ] T-money交通卡（或准备现金购买）
- [ ] 支付宝/微信（部分商户支持）
- [ ] 信用卡额度确认（用于购物）

## 电子设备

- [ ] 手机+充电器
- [ ] 充电宝（20000mAh以内可带上飞机）
- [ ] 韩国转换插头（德标/欧标）
- [ ] 插线板（如需多设备充电）
- [ ] 耳机
- [ ] 相机（如需，注意演唱会现场禁止专业设备）

## 应援物品

- [ ] 官方应援棒
- [ ] 7号电池（至少3-6组）
- [ ] 手幅/扇子
- [ ] 团体周边T恤/配饰
- [ ] 小卡（用于交换）
- [ ] 润喉糖（多备）
- [ ] 应援袋/小包

## 衣物与日用品

- [ ] 根据季节准备衣物
- [ ] 舒适的运动鞋（演唱会需站立/行走）
- [ ] 便携雨伞（查看天气预报）
- [ ] 防晒霜（夏季）
- [ ] 保湿护肤品（韩国冬季干燥）
- [ ] 常用药品（感冒药、肠胃药、创可贴）
- [ ] 卫生巾（如需要）
- [ ] 隐形眼镜+护理液（如需要）

## 通讯与导航

- [ ] 开通国际漫游/购买韩国流量卡
- [ ] 下载Naver Map
- [ ] 下载Kakao T（打车用）
- [ ] 下载Papago翻译App
- [ ] 保存酒店地址（韩文）
- [ ] 保存场馆地址（韩文）

## 出发前24小时

- [ ] 在线值机（如可）
- [ ] 确认航班状态
- [ ] 确认天气
- [ ] 给充电宝充满电
- [ ] 给应援棒换好新电池
- [ ] 备份重要文件到云端
- [ ] 告知家人/朋友行程
- [ ] 清空手机存储空间（拍照用）

## 机场出发当天

- [ ] 护照
- [ ] 机票/登机牌
- [ ] 手机
- [ ] 充电宝（随身带，不可托运）
- [ ] 应援棒（随身带，电池取出或单独包装）
- [ ] 韩元现金
- [ ] 信用卡
