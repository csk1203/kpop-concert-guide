## 📖 项目简介

本仓库是一套完整的 **K-pop 韩国演唱会追星指南 Skill**，专为中国粉丝设计。无论你是第一次去韩国看演唱会，还是已有经验但希望更系统地准备，这份指南都能提供从行前准备到现场应援的全流程帮助。

内容基于对中国粉丝常见问题（签证、抢票、语言、应援文化等）的深入调研，结合韩国演唱会实际场景编写，可独立阅读，也可作为 AI 助手（如 Claude Skill）的知识库使用。

---

## ✨ 特点

- **时效性强**：涵盖韩国主要票务平台、场馆、签证政策等最新信息
- **步骤清晰**：按时间顺序从签证→抢票→行前→现场→返程
- **多场景覆盖**：会员预售、公售、二手票、场馆交通、应援礼仪、紧急情况
- **清单化管理**：提供可直接打印的检查清单，防止遗漏
- **AI 友好**：结构化 Markdown，可作为 Skill 或 RAG 知识库

6. **行前检查** → [assets/checklist.md](assets/checklist.md) 和 [assets/packing-list.md](assets/packing-list.md)

### 方式二：作为 AI Skill 使用（推荐）
将本仓库内容导入支持 Skill 的 AI 助手（如 Claude），通过对话获取个性化建议：

- “我想明年 3 月去首尔看 SEVENTEEN 演唱会，需要提前多久办签证？”

### 方式三：本地搜索 / 打印
