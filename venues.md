## 场馆交通详解

### 高尺天空巨蛋（고척스카이돔）

**地址**：首尔特别市九老区京仁路 430

**交通方式**：
- **地铁1号线** 九一站（구일역）1号出口，步行约10分钟
- **地铁1号线** 新道林站可换乘

**从仁川机场**：
- 机场铁路 → 首尔站 → 地铁1号线 → 九一站
- 全程约1.5-2小时

**从明洞/弘大**：
- 明洞：地铁2号线 → 新道林站换乘1号线 → 九一站
- 弘大：地铁2号线 → 新道林站换乘1号线 → 九一站

**周边设施**：
- 场馆内有便利店和餐饮
- 附近有Homeplus超市
- 周边酒店较少，建议住弘大/明洞

### KSPO DOME（奥林匹克体操竞技场）

**地址**：首尔特别市松坡区奥林匹克路 424

**交通方式**：
- **地铁5号线** 奥林匹克公园站（올림픽공원역）3号出口
- **地铁9号线** 奥林匹克公园站
- 出站后步行约5-10分钟

**从仁川机场**：
- 机场铁路 → 金浦机场站换乘9号线 → 奥林匹克公园站
- 或机场大巴6006线直达

**周边设施**：
- 位于奥林匹克公园内，环境优美
- 公园内有餐饮和便利店
- 周边酒店：乐天世界酒店、肯辛顿酒店

### 蚕室综合运动场/蚕室室内体育馆

**地址**：首尔特别市松坡区奥林匹克路 25

**交通方式**：
- **地铁2号线** 综合运动场站（종합운동장역）6号/7号出口
- **地铁9号线** 综合运动场站

**从仁川机场**：
- 机场铁路 → 首尔站 → 地铁4号线 → 东大门历史文化公园站换乘2号线 → 综合运动场站
- 或机场大巴6006线

**周边设施**：
- 旁边有乐天世界、乐天塔
- 乐天百货、乐天玛特
- 住宿推荐：乐天世界酒店、Signiel Seoul

### SK奥林匹克手球竞技场

**地址**：首尔特别市松坡区奥林匹克路 25

**交通方式**：
- 与蚕室室内体育馆同站
- **地铁2号线/9号线** 综合运动场站

**特点**：
- 规模较小，氛围更亲密
- 适合粉丝见面会、小型演唱会

## 场馆周边信息

### 住宿推荐

**高尺天空巨蛋周边**：
- 九老区域酒店较少且偏商务
- 推荐住弘大（地铁30分钟）或明洞（地铁40分钟）

**KSPO DOME/蚕室周边**：
- 乐天世界酒店：高端，步行可达
- 肯辛顿酒店：中高端
- 松坡区商务酒店：性价比高
- 江南区酒店：地铁便利

**住宿预订建议**：
- 演唱会期间酒店涨价且紧张，提前2-3个月预订
- 选择可免费取消的房型以防计划变更

### 餐饮推荐

**高尺天空巨蛋**：
- 场馆内有Food Court
- 附近九老数码园区有餐饮街

**KSPO DOME**：
- 奥林匹克公园内有餐厅和咖啡厅
- 石村湖周边有很多美食

**蚕室**：
- 乐天世界购物中心美食丰富
- 石村湖美食街
- 松坡区本地餐厅

### 便利店与物资采购

**应援物资**：
- 电池：场馆周边便利店（应援棒用7号电池）
- 手幅/扇子：通常在场外有粉丝发放或售卖

**推荐便利店**：
- CU、GS25、7-Eleven
- 可购买T-money卡充值、零食饮料

## 入场流程

### 到达时间建议

| 活动 | 建议到达时间 | 说明 |
|------|--------------|------|
| 领取应援物 | 开场前4-5小时 | 站子 usually 提前发放 |
| 购买周边 | 开场前3-4小时 | 官方周边摊位通常很早开 |
| 入场 | 开场前1-2小时 | 留足安检和找座位时间 |

### 入场流程

1. **确认入口**：
   - 根据票面上的Gate/入口信息前往
   - 大型场馆可能有多个入口

2. **排队安检**：
   - 查票（电子票或实体票）
   - 查包（检查禁带物品）
   - 安全检查

3. **身份核对**（部分演出）：
   - 出示护照
   - 核对票上姓名

4. **入场**：
   - 根据座位号找到位置
   - 测试应援棒（如有场控）

### 禁带物品

**通常禁止**：
- 专业摄影摄像设备（单反、三脚架、自拍杆）
- 外食饮料（场馆内有售）
- 过大背包（建议小包或应援袋）
- 危险物品（打火机、激光笔等）

**可以携带**：
- 手机（但禁止录制）
- 小型充电宝
- 应援棒、手幅、扇子
- 小包装零食、润喉糖

**提示**：
- 每个场馆规定可能略有不同
- 关注演出前公告
- 存包服务通常收费

## 散场攻略

### 散场时间预估

- 演唱会通常2-3小时
- 加安可后可能延长
- 大型场馆散场需30-60分钟

### 交通返程

**地铁**：
- 末班车通常在午夜12点左右
- 建议提前查好末班车时间
- 散场人流大，可能需要排队进站

**出租车**：
- 场馆外通常有出租车等候区
- 可使用Kakao T叫车
- 深夜需支付夜间附加费

**夜间巴士**：
- 首尔有夜间巴士N系列
- 班次较少，需提前查询路线

### 散场注意事项

1. **保持秩序**：跟随人流，不要推挤
2. **看管财物**：人多时注意安全
3. **保持联络**：与同伴约定集合点
4. **应援物处理**：有些可带走作为纪念

### 深夜返回酒店

**安全建议**：
- 尽量结伴而行
- 使用正规出租车或App叫车
- 酒店偏远的建议提前约车
- 保持手机电量

## 特别提醒

**天气变化**：
- 韩国四季分明，户外演唱会注意天气
- 夏季注意防暑，冬季注意保暖

**场馆WiFi**：
- 大型场馆通常有免费WiFi
- 但人多时信号可能不佳
- 建议提前下载离线地图

**语言沟通**：
- 场馆工作人员基本英语可沟通
- 准备翻译App备用
- 重要信息截图保存
